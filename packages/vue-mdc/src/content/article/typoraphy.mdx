# Typography

## Required

Typography組件需要引入`typography`組件的樣式:

```css
@import url('@glare-labs/vue-mdc/build/components/typography/styles/typography.module.css');
```

Typography組件需要引入`typography`組件的核心:

```js
import { Button } from '@glare-labs/vue-mdc/button'
```
