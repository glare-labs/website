# IconButton

## Required

IconButton組件需要引入`icon-button`, `ripple`和`elevation`組件的樣式:

```css
@import url('@glare-labs/vue-mdc/build/components/icon-button/styles/icon-button.module.css');
@import url('@glare-labs/vue-mdc/build/components/ripple/styles/ripple.module.css');
@import url('@glare-labs/vue-mdc/build/components/elevation/styles/elevation.module.css');
```

IconButton組件需要引入`icon-button`組件的核心:

```js
import { IconButton } from '@glare-labs/vue-mdc/icon-button'
```

若要在Fab中使用圖標, 需要引入Icon組件樣式和核心:

```css
@import url('@glare-labs/vue-mdc/build/components/icon/styles/icon.module.css');
```

```js
import { Icon } from '@glare-labs/vue-mdc/icon'
```

|`"outlined"`|`EIconButtonAppearance.Outlined`|
|`"standard"`|`EIconButtonAppearance.Standard`|
