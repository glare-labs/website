# Button

## Required

Button組件需要引入`button`, `ripple`和`elevation`組件的樣式:

```css
@import url('@glare-labs/vue-mdc/build/components/button/styles/button.module.css');
@import url('@glare-labs/vue-mdc/build/components/ripple/styles/ripple.module.css');
@import url('@glare-labs/vue-mdc/build/components/elevation/styles/elevation.module.css');
```

Button組件需要引入`button`組件的核心:

```js
import { Button } from '@glare-labs/vue-mdc/button'
```

## Props

### Appearance

|Value|EButtonAppearance|
|:--|:--|
|`"filled"`|`EButtonAppearance.Filled`|
|`"filled-tonal"`|`EButtonAppearance.FilledTonal`|
|`"elevated"`|`EButtonAppearance.Elevated`|
|`"outlined"`|`EButtonAppearance.Outlined`|
|`"text"`|`EButtonAppearance.Text`|
